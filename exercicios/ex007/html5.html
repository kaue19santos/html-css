Aprenda a usar as tags

Para deixar o texto em negrito use as tags <strong> e </strong>, exemplo: Mano Zafe Mano Zafe, eu quero putaria

Para deixar o texto em itálico use as tags <em> e </em>, exemplo: Eu, tu, nós bota nela

Para deixar o texto marcado use as tags <mark> e </mark>, exemplo: Que irmã gostosa que esse maluco tem

Para deixar o texto pequeno use as tags <small> e </small>, exemplo: Me mandou embora da tua vida

Para deixar o texto riscado use as tags <del> e </del>, exemplo: Deus não está morto

Para deixar o texto sublinhado use as tags <ins> e </ins>, exemplo: Resgatei meu cartão black

Para deixar o texto embaixo de outro use as tags <sub> e <sub>, exemplo: H20

Para deixar o texto em cima de outro use as tags <sup> e </sup>, exemplo: 102
